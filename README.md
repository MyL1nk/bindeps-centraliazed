# USW Binary Dependencies

## Overview | Tổng quan

This repository is used by MyL1nk organization as the central place for managing binary dependencies, especially for Android development and its ecosystem.

Kho lưu trữ này được tổ chức MyL1nk sử dụng như một nguồn tập trung để quản lý các binary dependencies, chủ yếu phục vụ cho phát triển Android và các công cụ xoay quanh hệ sinh thái đó.

---

## What does it contain? | Repo này chứa gì?

- Android SDK, NDK, emulator tools
- Build, test, and code quality tools (e.g., lint, shellcheck)
- Adapters for database/API used in Android context
- CI/CD helper libraries and scripts
- Other tools and binaries commonly needed for Android projects

- Bộ công cụ SDK, NDK, emulator cho Android  
- Công cụ build, test, phân tích chất lượng mã nguồn (ví dụ: lint, shellcheck)  
- Các adapter/phần mềm hỗ trợ truy cập database, API trong môi trường Android  
- Thư viện hoặc script hỗ trợ quá trình CI/CD  
- Những công cụ nhị phân phổ biến khác, thường dùng cho dự án Android

---

## Purpose | Mục đích

To provide a unified, reliable source for Android-related binary dependencies across MyL1nk projects.  
This helps with version management, security, and avoiding external interruptions.

Cung cấp một nguồn tập trung, đáng tin cậy cho các binary dependencies liên quan đến Android trong toàn bộ các dự án của MyL1nk.  
Giúp thuận tiện quản lý phiên bản, bảo mật, giảm sự cố phụ thuộc vào nguồn ngoài.

---

## How to contribute? | Đóng góp như thế nào?

If you'd like to propose, add, or update binaries in this repository, please contact the repo administrators for review and approval.

Nếu bạn muốn góp ý, thêm mới hoặc cập nhật bất kỳ binary nào, vui lòng liên hệ quản trị viên repo để được xem xét và phê duyệt.

Nếu cần hỗ trợ hoặc có câu hỏi, hãy liên hệ qua kênh nội bộ tổ chức hoặc trực tiếp với những người quản lý repo.
